# Deepfake-Detection-System
📌 Project Name Deepfake Detection using AI & Machine Learning

📖 Description A Deepfake Detection System developed using Python and Machine Learning to identify manipulated images and videos. The project applies image preprocessing, feature extraction, and deep learning techniques to classify media as real or fake. It demonstrates the use of AI for digital media authentication and misinformation detection.

🚀 Features • Detects deepfake images and videos • Image preprocessing and feature extraction • Deep learning-based classification • User-friendly prediction interface • Accurate real vs. fake detection

🛠️ Technologies Used • Python • TensorFlow / Keras • OpenCV • NumPy • Pandas • Matplotlib • Scikit-learn (if used)

📂 Dataset Celeb-DF , FF++

▶️ How to Run

1.Clone the repository.

2.Install dependencies using pip install -r requirements.txt.

3.Run the application.
